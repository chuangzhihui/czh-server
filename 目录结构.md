 # 开发说明
 ## 1.目录结构
    czh/
    ├──admin/                                   # 后台管理模块
    │   ├──baseController                       # 后台基础控制器，不要随意修改此目录文件
    │   ├──controller                       # 项目开发的控制器存放目录
    │   ├──config/                              # 模块配置
    │   │       ├──CorsFilter                   # 自定义跨域配置
    │   │       ├──LoggingAspect                # 切面记录项目请求日志
    │   │       └──SecurityConfig               # spring-security配置
    │   └──security                             # spring-security 相关文件
    │           ├──AdminAuthFailedHandler       # 授权失败的回调
    │           ├──AdminDetails                 # 自定义SpringSecurity的user
    │           └──JWTAuthFilter                # 自定义JWT验证器
    ├──common/                                  # 公共基础模块
    │   ├── annotation                          # 项目自定义注解
    │   │       ├──Log                          # 配置后台请求方法是否需要记录日志
    │   │       └──Permission                   # 配置后台请求方法是否需要进行权限校验及关联的权限列表
    │   ├── config                              # 项目公共配置
    │   │       ├──CZHPasswordEncoder           # 自定义springSecurity 加密工具
    │   │       ├──ExecutorConfig               # 异步任务配置
    │   │       └──RedisConfig                  # Redis配置
    │   ├── dto                                 # 项目公共配置
    │   │    └──PageDto                         # 通用分页请求
    │   ├── exception                           # 自定义异常类
    │   ├── utils                               # 自定义助手类
    │   └── vo                                  # 公共返回类
    ├── service/                                # 数据服务类
    │   ├── dao   
    │   ├── dto                                 # 请求实体
    │   ├── entity                              # 数据实体
    │   ├── repository                          # mongoDb
    │   ├── service                             # 服务类
    │   └── vo                                  # 返回实体
    └── README.md                               # 说明文档

## 2.开发规范
    2.1 数据源必须从service中获取
    2.2 除数据表名外所有命名必须使用驼峰命名，数据表名使用下划线
